CU CS AMA Helpful Links

Here is a list of helpful websites a resources around the Interwebs! Check them out for a wide range of CS hints, tips, tricks, and help. For more information on the CU CS AMA Corps, see our homepage.

Help Forums

These websites are good places to ask for help, or to view answers to other people's questions. Take some time to learn the etiquette on each site before posting a new question. In general, it's always best to search through the site for your answer before posting a new question. Asking questions that have already been answered (and that a quick search would have revealed) is frowned upon.

  • Stack Overflow - Programming questions of all flavors
  • Server Fault - System administration questions of all flavors
  • Super User - General computer questions of all flavors
