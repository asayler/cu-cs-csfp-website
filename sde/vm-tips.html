Getting the Most Out of the VM

To ensure that your VM experience is as good as possible, please consider the tips, tricks, and recommendations contained on this page.

Backing Up Your Data

The VM, while fairly reliable, does tend to be more prone to failure and data loss then a physical computer due to the nature of VMs in general. Thus, it is important to ensure that you have backups of all personal data stored on the VM. There are a variety of methods for backing up data from the VM, several of which are listed below.

  • Dropbox: Dropbox is installed on the VM by default. If you set up a Dropbox account (or link the VM to an existing Dropbox account), any files you store in the '~/Dropbox/' directory will be automatically synced to the cloud. As long as you store all of your personal files in your Dropbox directory, using Dropbox can provide both a backup for your data on the VM as well as a means for transferring data on and off the VM.
  • Github: Github is a cloud-hosted Git repository. It is free to use and offers a simple way to backup and sync your code to the cloud (with all of the benefits of the Git SCM). If you place your code in Git repositories on the VM and then sync these repositories to your Github account, you can ensure that you always have a backup copy of your code off the VM.
