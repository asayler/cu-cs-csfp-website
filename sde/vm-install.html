Installing the VM

The CU CS Virtual Machine image is designed to run with Oracle VirtualBox. The VM itself runs Ubuntu 14.04 GNU/Linux. These directions will take you through the process of setting up the CU CS VM on your machine.

If you would like in-person help installing the VM, please contact your instructor, TA, or CSFP Personnel.

Before continuing, you should confirm that your system meets the system requirements.

You should also have obtained a copy of the VM image.

Install VirtualBox

To start, install VirtualBox on your computer. The method for doing this varies depending on your computer's OS:

Video: Installing Oracle VirtualBox - Windows 7 (Note: You have already downloaded VirtualBox; skip that step in this video)

  • If you are using Windows, run the previously downloaded 'VirtualBox-4.3.exe' file to install VirtualBox.
  • If you are using OSX, run the previously downloaded 'VirtualBox-4.3.dmg' file to install VirtualBox.
  • If you are using Linux, you can install VirtualBox via your system's package manager. See VirtualBox for Linux hosts.

Import the VM Image

Once VirtualBox has been installed, you are ready to import the VM image onto your machine.

Video: Importing the CU CS VM Image into Oracle VirtualBox

  1. Start by openeing VirtualBox
  2. From the main VirutalBox menu, select "File>Import Appliance". This will launch the "Appliance Import Wizard"
  3. Click the "Choose..." button on the first page of the wizard and select the previously downloaded .ova VM image file.
  4. Click "Next" to advance to the second page of the import wizard.
  5. Click "Import" to beging teh VM import process.
  6. Wait for the import to complete. It may take 5 to 15 minutes.

Run the VM

Once the import has completed, you are ready to run the VM. Repeat this process each time you wish to use the VM.

Video: Booting and Using the CU CS Virtual Machine

  1. Select the VM in the main VirtualBox window, and click the green "Start" arrow near the top of the screen. This will launch the VM in a new window.
  2. The VM will boot much like a regular computer does. Depending on your system, you may be presented with several VirutalBox dialog boxes during the boot process. These are generally informational and may be dismissed once read. (You may wish to check the "Do not show again" option to avoid dealing with these dialogs each time your boot the VM.)
  3. Once booted, you should be presented with the Ubuntu 14.04 desktop inside the VM window. The desktop should automatically re-size itself to fit the VM window. If it does not, you may wish to try clicking and dragging the corner of the VM window to change its size. Often, you must re-size the VM window once or twice to before the desktop auto-resizing will begin to work.
  4. The first time you boot the VM, you will be presented with the "Dropbox Setup" wizard on the Ubuntu desktop. Dropbox is an easy way to get files on and off your VM, as well as to provide you with a cloud-based backup of your files. If you wish to use Dropbox, follow the wizard to complete the setup process. Using Dropbox is strongly encouraged to ensure you always have a cloud-based backup of your VM work.
  5. At this point, your VM is ready to use. The most commonly needed programs are all available via shortcuts on the left-hand sidebar. Play around a bit to familiarize yourself with Ubuntu.
  6. When you are done using your VM, shut it down in the same manor you would a physical machine. In Ubuntu, this involves clicking the small "gear" icon in the upper right corner of the screen. Once the VM shutdown, the VM window will automatically close.

Additional VM tips and tricks, as well as default VM credentials, can be found on the Getting the Most Out of Your VM page.

Addition information on VirtualBox can be found in the VirtualBox User Manual.

Additional information on Ubuntu can be found on the Ubuntu website.
