Downloading the VM Files

We recommend downloading the VM files via the BitTorrent peer-to-peer distribution system. For those that can not use BitTorrent, we also provide a direct download method. Note, however, that the direct download method is slower and more prone to corruption than the BitTorrent method.

Older versions of the VM can be found on the CSEL Archive site.

BitTorrent Download

Downloading the VM files via BitTorrent involves downloading a '.torrent' file that you load using a BitTorrent client to download the actual VM files.

If you do not already have a BitTorrent client installed, please install one from the list below:

  • qTorrent (OSX + Windows + Linux, Free)
  • Deluge (OSX + Windows + Linux, Free)
  • Transmission (Linux + OSX, Free)

Next, download the torrent file below:

VM Torrent (includes VM image, VirtualBox installers, and checksums): cu-cs-vm-fall14-x64-v1.1.torrent

Open the torrent file in your BitTorrent client to begin your high-speed download of the VM files.

Once the download completes, proceed to the next step.

Direct Download

If you can not use BitTorrent to download the VM files, you may instead use the direct download links below:

VM ZIP: (includes VM image, VirtualBox installers, and checksums): cu-cs-vm-fall14-x64-v1.1.zip

Once the download completes, extract the zip file and proceed to the next step.

Low-Graphics Version

The standard VM (above) works best with machines that support virtualized 3D acceleration. For machines that do not support virtualized 3D acceleration, we offer a "low-graphics" version of the VM as an alternative. This VM uses the XFCE desktop environment. It thus looks a bit different than the standard VM, but it contains all the same software and functionality.

Like the standard VM, we offer the low-graphics VM via either torrent or direct download (torrent recommended):

Low-Graphics VM Torrent (includes VM image, VirtualBox installers, and checksums): cu-cs-vm-fall14-x64-xfce-v1.1.torrent Low-Graphics VM ZIP: (includes VM image, VirtualBox installers, and checksums): cu-cs-vm-fall14-x64-xfce-v1.1.zip

Once the download completes, follow the normal installation instructions in the next step to setup the low-graphics VM.
