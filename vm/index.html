CU CS Virtual Machine

This website hosts information and files related to the University of Colorado Department of Computer Science Virtual Machine. The CU CS VM is part of the CU CS Foundation Program. A Virtual Machine is a simulated "computer" that runs as a program on your real computer. We use the VM to provide students with a uniform, officially-supported, Linux-based development environment.

System Requirements

To run the VM effectively on your own machine, your computer should meet the following requirements:

  • 4GB of RAM or more
  • Intel Core2 Due or newer processor with virtualization and 64-bit support
    (Intel Core i3 or newer recommended; Intel Atom processors NOT recommended)
  • Processor virtualization support enabled at the BIOS/UEFI level
    (Process varies by model, See this video for an example)
  • 2 or more physical processor cores
  • 30GB or more of free HD space
  • A 64-bit OS (Windows 7 or newer, OSX 10.7 or newer, etc)

Downloading the VM

The recommended VM download method is via the BitTorrent peer-to-peer distribution system. For those that can not use BitTorrent, we also provide a direct download method. Note, however, that the direct download method is slower and more prone to corruption than the BitTorrent method.

Each download also includes a link for the md5 hash file associated with the VM image. You can (optionally) use these hash files with the Unix 'md5sum' utility to verify that your download is not corrupt. See this link for more information.

Download via BitTorrent

Downloading the VM via BitTorrent involves downloading a '.torrent' file that you can then open with a BitTorrent client to download the actual VM file. Using BitTorrent will help ensure you the fastest download, will ensure your download is not corrupted, and will allow you to easily start, stop, a resume your download should you need to. If you do not already have a BitTorrent client installed, please install one from the list below:

  • Transmission (Linux + OSX, Free)
  • FrostWire (Linux + OSX + Windows, Free)
  • qTorrent (Linux + OSX + Windows, Free)

Once you have installed a BitTorrent client, you may download the one of the .torrent files for the VM below. Open the .torrent file in your BitTorrent client to begin your VM download.
