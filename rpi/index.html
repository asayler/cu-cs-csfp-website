CU CSFP Raspberry Pi Project

The CSFP Raspberry Pi project aims to promote and support the use of the Raspberry Pi Linux Microcomputer. The RPi Project is made possible with the help of the College of Engineering and Applied Science via a Spring 2013 EEF mini grant.

Equitment

The basic CSFP RPi kit consits of the following components:

  • 1 - Raspberry Pi Model B Microcomputer (Available from MCM, Adafruit, Newark, and Allied)
  • 1 - SanDisk Extreme 8GB SDHC Flash Memory Card (SDSDX-008G, Available from Amazon, Etc)
  • 1 - Airlink Wireless 150 Ultra Mini-USB Adapter (AWLL5099, Available from Amazon, Etc)
  • 1 - Micro USB to USB Cable (For charging/power, Available from Amazon, Etc)
  • 1 - HDMI to DVI Cable (OPTIONAL - For display on DVI monitor, Available from Amazon, Etc)
  • 1 - Piface Digital IO Expansion Board (OPTIONAL - For external connectivity, Available from MCM and Newark)
