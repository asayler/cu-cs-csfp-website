A Raspberry Pi Web App

This page contains instructions for programming and hosting a small web app on your Raspberry Pi in the context of the CU CSFP RPi Project. This page builds on the previous two classes: Programming on the RPi and Setting up the RPi as a Server.

Prerequisites

We will be starting with the the Raspbain OS configured as we left it at the end of the third class. If you are using your own RPi or SD Card, you may download a fresh version (sha1) of this configuration and flash it via the normal means. You will need to manually configure the network as discussed in the previous class, but Apache has already been installed for you.

Code

The code is available on GitHub via the TwitterAlarmClock repository. See the 'web-cgi' branch.
