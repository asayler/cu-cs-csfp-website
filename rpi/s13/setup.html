Setting Up Your Raspberry Pi

This page contains instructions for setting up and configuring your Raspberry Pi in the context of the CU CSFP RPi Project. These instructions will be covered the first RPi project class. In addition, the Raspberry Pi Foundation also provides an illustrated "Getting Started" guide. The elinux.org RPi Wiki is also a good general reference source.

Step 1: SD Card Setup

This step is covered in the "Setup and Config - Part 02: SD Card Setup" video segment on YouTube.

The first step to setting up the RPi is installing an operating system image to the RPi's SD card. We will be using the Raspbain OS for this course. This OS is a Debian Linux derivative and is the official recommended OS for the RPi.

First, download the latest Raspbain image: http://www.raspberrypi.org/downloads.

Next, follow the directions to copy the Raspbain image to your SD card: http://elinux.org/RPi_Easy_SD_Card_Setup.

Step 2: Connecting the Pi

This step is covered in the "Setup and Config - Part 03: Connecting the RPi" video segment on YouTube.

Connect the pi to the keyboard, video, and monitor. Insert the prepared SD card and USB WiFi adapter (if present). Do not connect the micro-USB power cable until everything else is connected and you are ready to boot.

Step 3: Initial Boot

This step is covered in the "Setup and Config - Part 04: First Boot" video segment on YouTube.

Connect the USB power cable to boot the RPi. You should be presented with the initial blue-background configuration 'raspi-config' screen. If not, you may run 'raspi-config' manually.

We will use the 'raspi-config' tool to configure several items:

  • Grow SD Partition (expand_rootfs)
  • Configure Keyboard: 'Generic 104-key PC' -> Other -> English (US) -> Default -> Default -> Yes
  • Change Password: YOU PICK
  • Change Locale: en_US.UTF-8 (select via Space-bar) -> Set Default to en_US.UTF-8
  • Change Timezone: America -> Denver
  • Set GPU/CPU Memory Split: 64 MB for GPU
  • Set SSH: Enable
  • Boot to Desktop: No

Finally, reboot to complete the initial setup. The system should come back up to a CLI log-in prompt. The default user it 'pi' and the default password is 'raspberry' (unless you changed it as mentioned above).

Step 7: Test SSH

This step is covered in the "Setup and Config - Part 05: The Desktop" video segment on YouTube.

Find IP Address and Log-in via SSH

On RPI:

'ifconfig' (look for either eth0 or wlan0 IP address)

On another machine (must be on same network as RPi):

'ssh USER@IP' (Linux and OSX)

Windows user should use Putty.
